# native

## 手写原生方法

- 目的有三:
  1. 充分理解方法的使用
  2. 理解js内部执行原理,提升js编程水平
  3. 提升面试水品,面试必考知识点

- 介绍
  1. 每个文件表示一个方法或功能的实现
  2. 文件名表示写作日期
  3. 想查阅某个方法的实现可以看以下目录
  4. 每个方法实现我会在源码中详细注释
  5. 自定义的方法名和原生只差一个'```_```'开头
